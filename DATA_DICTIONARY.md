# Data Dictionary

## Fields

| Field Name | Type | Description |
|---|---|---|
| Product_ID | Integer | Unique product identifier |
| Product_Name | Text | Product name |
| Brand | Text | Cosmetic brand name |
| Category | Text | Product category such as foundation, lipstick, cleanser, or serum |
| Price | Decimal | Product price |
| Rating | Decimal | Average customer rating |
| Reviews | Integer | Number of customer reviews |
| Sales_USD | Decimal | Sales value in USD |
| Skin_Type | Text | Suitable skin type |
| Ingredient_Type | Text | Main ingredient or product feature |
| Launch_Year | Integer | Year the product was launched |
| Country | Text | Market or country associated with the product |
