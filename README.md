# Cosmetic Insights
Open live dashboard: [Click here](https://nishadafsheen-source.github.io/Cosmetic-Insights-Navigating-Cosmetic-Trends-and-Customer-Insights-with-Tableau/)

Cosmetic Insights is a Tableau-based data analytics project that examines cosmetics data to uncover market trends, consumer preferences, brand performance, pricing patterns, and skin-type suitability. The project uses interactive dashboards to present actionable insights that support data-driven business decisions and product strategy.

## Project Goal

Identify what customers buy, which brands perform best, how pricing affects demand, and which product types are best suited to different skin types.

## Business Questions

- Which product categories generate the strongest interest and sales?
- Which brands outperform competitors on ratings and sales?
- How do pricing patterns vary across categories and brands?
- Which ingredients or product types are most suitable for specific skin types?
- What trends can guide marketing and product strategy?

## Deliverables

- Tableau dashboard plan
- Cosmetics sample dataset
- Data dictionary
- Project report
- Tableau build guide
- Executive summary
- Presentation notes

## Key Metrics

- Total sales
- Average rating
- Average price
- Product count by category
- Brand performance by sales and rating
- Skin-type suitability distribution
